# 火山引擎（字节跳动）部署静态网站步骤

火山引擎提供 **对象存储 + CDN** 的方式托管静态网站，成本很低，国内访问速度快。

## 方案选择

火山引擎托管静态网站常用方案：
**对象存储 TOS → 配置静态网站 → 绑定 CDN 加速**

## 步骤

### 1. 注册/登录火山引擎
打开 https://www.volcengine.com/ 注册账号登录。

### 2. 创建对象存储桶

1. 进入「对象存储 TOS」控制台
2. 点击「创建桶」
3. 配置：
   - **存储桶名称**：比如 `poetry-website`（全局唯一）
   - **地域**：选离你近的（比如华北、华东）
   - **访问权限**：选择 `公共读`（因为网站要被人访问）
   - 其他默认即可，点击创建

### 3. 上传网站文件

创建完桶后，进入桶，点击「上传文件」：

把本地这几个文件上传上去：
- `index.html`
- `style.css`
- `script.js`
- `data.js`

可以直接拖进去上传。

### 4. 开启静态网站功能

1. 在桶列表点击你的桶 → 点击「属性」标签
2. 找到「静态网站」，点击「编辑」
3. 开启「静态网站」
4. 配置：
   - **默认首页**：`index.html`
   - **默认404页**：也可以填 `index.html`（单页应用需要，我们静态站也可以）
5. 保存

保存后会给你一个 **默认访问地址**，类似：
```
http://poetry-website.s3-cn-beijing.volces.com
```

现在你已经可以通过这个地址访问网站了！

### 5. （可选）绑定自定义域名 + CDN 加速

如果要加速、要自定义域名：

1. 进入「内容分发 CDN」控制台
2. 点击「添加域名」
3. 配置：
   - **域名**：填你自己的域名（比如 `poetry.example.com`）
   - **源站信息**：选择「火山引擎 TOS」，选择你刚才的桶
   - **加速区域**：选择「中国大陆」
   - 其他默认，提交
4. 按照提示去你的域名服务商那里改 DNS
5. 等待生效（一般几分钟到几小时）

## 自动化部署（可选）

安装火山引擎 CLI，可以命令行一键上传：

```bash
# 安装 CLI
curl -fsSL https://artifact.bytedance.com/release/volcengine-cli/install | bash

# 配置 AK/SK（在火山引擎控制台获取）
volcengine-cli configure set --access-key your-ak --secret-key your-sk --region cn-beijing

# 批量上传当前目录文件到桶
# 可以用这个脚本一键部署
for file in *.html *.css *.js; do
  volcengine-cli tos cp $file tos://你的桶名/$file --acl public-read
done
```
